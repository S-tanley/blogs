Statistics---Likelihood Ratio Test: From Simple Hypothesis to Composite Hypothesis

Created2025-11-18|Updated2025-11-18|Study Blogs
|Post Views:

本篇内容主要就来讲一讲 likelihood ratio，感觉这部分在大部分的理工科统计中是没有被覆盖的，但却是一个有关 hypothesis test 的重要内容。

一、 简单假设下的似然比检验 (Simple Hypothesis)

似然比检验 (Likelihood Ratio Test, LRT) 提供了一种构造检验统计量（test statistic）的通用方法。首先考虑最简单的情况：检验两个完全指定了概率分布的假设。

1. 定义似然比

假设观测数据 X⃗=(X1,…,Xn)\vec{X} = (X_1, \dots, X_n)X=(X1​,…,Xn​) 具有联合概率密度函数或频率函数 f(x∣θ)f(x | \theta)f(x∣θ)。我们设定两个简单假设：

  • 原假设 (H0H_0H0​)： X⃗∼f0(x⃗)\vec{X} \sim f_0(\vec{x})X∼f0​(x)
  • 备择假设 (HAH_AHA​)： X⃗∼fA(x⃗)\vec{X} \sim f_A(\vec{x})X∼fA​(x)

在观测到数据 x⃗\vec{x}x 后，似然比统计量 LRLRLR 定义为：

LR(x⃗)=L(θ0∣x⃗)L(θA∣x⃗)=f0(x⃗)fA(x⃗)LR(\vec{x}) = \frac{L(\theta_0 | \vec{x})}{L(\theta_A | \vec{x})} = \frac{f_0(\vec{x})}{f_A(\vec{x})} LR(x)=L(θA​∣x)L(θ0​∣x)​=fA​(x)f0​(x)​

2. 最优性：Neyman-Pearson 引理

Neyman-Pearson 引理证明了基于似然比构造的检验是最优的。最优性是指在固定第一类错误概率 α\alphaα（显著性水平）时，似然比检验能够提供最大功效 (1−β1-\beta1−β)。

  • LRT 决策规则 d(x⃗)d(\vec{x})d(x)： 如果似然比 LR<cLR < cLR<c，则拒绝 H0H_0H0​。

    dLRT(x⃗)={1if LR<c  ⟹  x⃗ prefers HA (Reject H0)0if LR≥c  ⟹  x⃗ prefers H0 (Fail to reject H0)d_{\text{LRT}}(\vec{x}) = \begin{cases} 1 & \text{if } LR < c \implies \vec{x} \text{ prefers } H_A \text{ (Reject } H_0 \text{)}\\ 0 & \text{if } LR \ge c \implies \vec{x} \text{ prefers } H_0 \text{ (Fail to reject } H_0 \text{)}\end{cases} dLRT​(x)={10​if LR<c⟹x prefers HA​ (Reject H0​)if LR≥c⟹x prefers H0​ (Fail to reject H0​)​

  • 结论： 在 α\alphaα 水平下，LRT 是功效最大的检验。

二、 推广到复合假设 (General Case)

在大多数实际问题中，假设是复合假设（Composite Hypothesis）。在这种情况下，我们使用广义似然比检验 (Generalized Likelihood Ratio Test, GLRT)。

1. 广义似然比统计量 (Λ)(\Lambda)(Λ)

假设 Ω\OmegaΩ 是总参数空间，ω0\omega_0ω0​ 是原假设下的参数子空间。广义似然比统计量 (Λ)(\Lambda)(Λ) 的定义为：

Λ=max⁡θ∈ω0[lik(θ)]max⁡θ∈Ω[lik(θ)]\Lambda = \frac{\max_{\theta \in \omega_0} [\text{lik}(\theta)]}{\max_{\theta \in \Omega} [\text{lik}(\theta)]} Λ=maxθ∈Ω​[lik(θ)]maxθ∈ω0​​[lik(θ)]​

  • 取值范围： 0≤Λ≤10 \le \Lambda \le 10≤Λ≤1。
  • 拒绝域： 小的 Λ\LambdaΛ 值支持备择假设，故拒绝域形式为 Λ≤λ0\Lambda \le \lambda_0Λ≤λ0​。

2. 拒绝域的确定

临界值 λ0\lambda_0λ0​ 的选择需要保证检验的显著性水平为 α\alphaα：

P(Λ≤λ0∣H0)=αP(\Lambda \le \lambda_0 | H_0) = \alpha P(Λ≤λ0​∣H0​)=α

3. 渐近分布：Wilk’s Theorem

对于大样本量，Wilk’s Theorem 提供了 −2log⁡Λ-2 \log \Lambda−2logΛ 的渐近分布：

  • 定理： 在适当的条件下，当样本量 n→∞n \to \inftyn→∞ 时，统计量 −2log⁡Λ-2 \log \Lambda−2logΛ 的空分布趋近于一个卡方 (χ2\chi^2χ2) 分布。

  • 自由度： 自由度 ddd 等于参数空间维度的差值：

    d=dim⁡(Ω)−dim⁡(ω0)d = \dim(\Omega) - \dim(\omega_0) d=dim(Ω)−dim(ω0​)

A.1 Neyman-Pearson 引理的核心论证

引理： 对于给定的显著性水平 α\alphaα，基于似然比 LR(x⃗)=f0(x⃗)fA(x⃗)LR(\vec{x}) = \frac{f_0(\vec{x})}{f_A(\vec{x})}LR(x)=fA​(x)f0​(x)​ 构造的检验 d(x⃗)d(\vec{x})d(x) 是最优检验（功效最大）。

  1. 定义 LRT 决策函数 d(x⃗)d(\vec{x})d(x)：

    对于任何其他决策规则 d∗(x⃗)d^*(\vec{x})d∗(x)，必须满足显著性水平约束：

    P(d∗(X⃗)=1∣H0)≤P(d(X⃗)=1∣H0)=αP(d^*(\vec{X})=1|H_0) \le P(d(\vec{X})=1|H_0) = \alpha P(d∗(X)=1∣H0​)≤P(d(X)=1∣H0​)=α

    E0[d∗(x⃗)]≤E0[d(x⃗)]E_0[d^*(\vec{x})] \le E_0[d(\vec{x})] E0​[d∗(x)]≤E0​[d(x)]

    对于所有的 x⃗\vec{x}x，根据 d(x⃗)d(\vec{x})d(x) 的构造方式，以下不等式成立：

    d∗(x⃗)⋅[cfA(x⃗)−f0(x⃗)]≤d(x⃗)⋅[cfA(x⃗)−f0(x⃗)]d^*(\vec{x}) \cdot [c f_A(\vec{x}) - f_0(\vec{x})] \le d(\vec{x}) \cdot [c f_A(\vec{x}) - f_0(\vec{x})] d∗(x)⋅[cfA​(x)−f0​(x)]≤d(x)⋅[cfA​(x)−f0​(x)]

A.2 Wilk’s Theorem 渐近分布的原理

Wilk’s Theorem 阐述了 −2log⁡Λ-2 \log \Lambda−2logΛ 的渐近分布为 χ2\chi^2χ2 分布，自由度为 dim⁡(Ω)−dim⁡(ω0)\dim(\Omega) - \dim(\omega_0)dim(Ω)−dim(ω0​)。

启发式解释：

−2log⁡Λ-2 \log \Lambda−2logΛ 统计量渐近等价于皮尔逊卡方统计量，其中皮尔逊卡方统计量本身是 mmm 个正态随机变量平方和的近似，而 mmm 个正态随机变量的平方和服从卡方分布。

具体来说，通过将对数似然函数在无约束 MLE 处进行泰勒级数展开，可以证明：

∑i=1mxilog⁡(xi/npi(θ^))≈12n∑i=1m(xi−npi(θ^))2pi(θ^)\sum_{i=1}^{m} x_i \log \left(\frac{x_i/n}{p_i(\hat{\theta})}\right) \approx \frac{1}{2n} \sum_{i=1}^{m} \frac{(x_i - n p_i(\hat{\theta}))^2}{p_i(\hat{\theta})} i=1∑m​xi​log(pi​(θ^)xi​/n​)≈2n1​i=1∑m​pi​(θ^)(xi​−npi​(θ^))2​

左侧的 2n2n2n 倍即为 −2log⁡Λ-2 \log \Lambda−2logΛ。因此，−2log⁡Λ-2 \log \Lambda−2logΛ 渐近等价于右侧的皮尔逊卡方统计量 χ2\chi^2χ2，其自由度反映了模型在约束解除后增加的自由参数数量。
